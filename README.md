## Corn Yield Benchmark SQL Project

## Overview
Intermediate SQL project built around agriculture and agribusiness analysis.

## The project compares a small set of farm-level corn records against a **real USDA national corn yield benchmark** for 2020-2024. The benchmark table is sourced from USDA National Agricultural Statistics Service annual crop production releases. Farm-level fertilizer and rainfall fields were added to create decision-oriented SQL questions.

## Main business question
Which farms are outperforming or lagging a real national corn-yield benchmark, and what input patterns might explain the gap?
Source note

## USDA benchmark values included:
2020 = 172.0 bu/acre
2021 = 177.0 bu/acre
2022 = 173.3 bu/acre
2023 = 177.3 bu/acre
2024 = 179.3 bu/acre
